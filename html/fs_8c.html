Подробное описание

Консольная утилита fs предназначена для создания виртуального диска и добавления на него файлов. Пример создания виртуального диска:

Автор
zergon321 (maxim.nosp@m.grad.nosp@m.an@gm.nosp@m.ail..nosp@m.com) fs create -o disk -s 1024

Данная команда создаёт виртуальный диск disk размером 1024 блока. Для записи файла на диск необходимо написать следующее:

fs write -d disk -f file.txt

Данная команда записывает на виртуальный диск disk файл file.txt. Каждый файл на виртуальном диске будет занимать целое число блоков.
